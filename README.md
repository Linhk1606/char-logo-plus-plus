# CharLOGO++

## 概述——这是啥
* 一个可以帮你**生成字符画LOGO**的沙雕玩意儿
* 使用 **C++** 编写
* **VSCode** 编辑器（看到没有，都帮你配置好JSON了，直接Debug就行了）原来是用CLion，然鹅不太习惯CMake，所以就抛弃了
* 至少在 `Ubuntu 19.04 x64` 上跑是没有问题的
* 注释写的很少，码风还可以（被 VSCode 格式化过）但是看懂我代码估计困难，以后有时间多写些注释
